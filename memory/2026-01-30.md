## Key Learnings
- **Hetzner firewall is outside the OS** ‚Äî sshd config can't fix provider-level blocks
- **ExpressVPN is terrible for SSH** ‚Äî blocks/rate-limits port 22, dynamic IPs cause lockouts
- **Tailscale is the correct answer** ‚Äî private mesh, no IP lockout risk, works with VPN off
- **100.x.x.x IPs:** 
  - 100.64.0.0/10 = CGNAT (Hetzner private network)
  - 100.x.x.x in Tailscale = also private but different space
- **User setup:** SSH key must exist in `~/.ssh/authorized_keys` with correct perms (600, owned by user)
