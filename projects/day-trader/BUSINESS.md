# BUSINESS.md - Fake Day Trader Project

## Project Overview
**Name:** Fake Day Trader
**Type:** Simulation/learning project
**Goal:** Build automated trading strategies, track fake portfolio, learn market patterns
**Status:** Active development

## Context
- **Not real money** - Pure simulation using market data
- **Learning focus** - Understand trading strategies, risk management, automation
- **Tools first** - Build infrastructure that could scale to real trading later
- **Shirin's interest** - Finance tracking + automation + data visualization

## Tech Stack
- **Data sources:** Yahoo Finance API, Alpha Vantage (free tier)
- **Analysis:** Python/pandas, technical indicators (RSI, MACD, moving averages)
- **Automation:** Clawdbot skills for monitoring, alerts, execution
- **Visualization:** Simple charts, Telegram updates, dashboard

## Current Phase
**Phase 1:** Infrastructure setup
- Market data collection (daily)
- Basic portfolio tracking
- Simple buy/sell simulation

**Phase 2:** Strategy development
- Technical indicator integration
- Backtesting framework
- Risk management rules

**Phase 3:** Automation
- Scheduled market scans
- Telegram alerts
- Automated "trades" (simulated)

## Portfolio (Example)
- **Starting capital:** $10,000 (fake)
- **Current holdings:** TSLA, AAPL, MSFT, GOOGL
- **Strategy:** Long-term growth + swing trading
- **Risk tolerance:** Medium (simulation only)
